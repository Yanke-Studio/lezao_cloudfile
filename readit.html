乐造文件云 - Lz's Studio

Python 3.9+ Flask 2.0+ MIT License

基于Flask开发的轻量级文件管理系统，提供安全的文件存储和分享服务

项目简介

乐造文件云是一个轻量级的文件管理系统，主要特点包括：

  • 采用邀请码注册机制，保障系统安全性
  • 支持文件上传、下载和在线预览
  • 提供完善的文件审核流程
  • 具备消息通知和系统公告功能
  • 详细的操作日志记录

功能特性

用户管理

  • 邀请码注册登录系统
  • 管理员/普通用户分级权限
  • 操作日志记录

文件管理

  • 文件上传与下载
  • 文件审核机制
  • 上传密码保护

消息系统

  • 用户消息通知
  • 管理员全局公告

安装部署

环境要求

  • Python 3.9+
  • Flask

安装步骤

1. 克隆或下载仓库

2. 推荐创建虚拟环境

python -m venv venv
# Windows
venv\Scripts\activate
# Linux/MacOS
source venv/bin/activate

3. 安装依赖

pip install flask

6. 启动应用

python app.py
